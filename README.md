# Controle de Estoque

Sistema de controle de estoque desenvolvido em Python com Orientação a Objetos.

## Tecnologias

* Python
* SQLite
* Git/GitHub
* SMTP Gmail

## Conceitos de POO Utilizados

* Encapsulamento
* Herança
* Polimorfismo
* Abstração

## Funcionalidades

* Cadastro de produtos
* Atualização de estoque
* Exclusão de produtos
* Consulta de produtos por setor
* Solicitação de item para uso
* Reposição automática de estoque
* Geração de solicitação de compra
* Envio automático de e-mail para fornecedor
* Persistência em banco SQLite
